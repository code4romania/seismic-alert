# API

## Buildings

- `GET` `/api/v1/buildings/`

```json
[
    {
        "general_id": 75,
        "risk_category": "U1",
        ...
    },
    {
        "general_id": 74,
        "risk_category": "U3",
        ...
    },
	...
]
```

{
